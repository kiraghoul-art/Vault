Como usar
  1. Abre o Spotify → clica nos três pontos da música/playlist → Partilhar → Copiar link
  2. Cola o link na zona de drop acima — a info é preenchida automaticamente
  3. Para o GIF: no Discord, clica com botão direito no GIF → Copiar endereço da imagem
  4. O link deve começar com media.discordapp.net ou media.tenor.com

Spotify API

Credenciais para o auto-fetch dos links. Usa a conta com Premium se quiseres trocar.

Como obter as credenciais
  1. Vai a developer.spotify.com/dashboard
  2. Cria uma app ou abre a existente Kira Vault
  3. Copia o Client ID e o Client Secret
  4. Em Redirect URIs certifica que tens: https://kira-discord-proxy.ghoullkira.workers.dev/spotify/callback
  5. Para usar a conta Premium: faz login nessa conta no Spotify e obtém as credenciais dessa app

Channels

Channels do Discord para backup de conteúdo. Podes adicionar, editar e remover.

Como obter um Channel ID
  1. No Discord, vai a Configurações → Avançado → Activa Modo de Programador
  2. Clica com botão direito no channel → Copiar ID do Channel
  3. O label é o nome que queres dar (ex: music, notas)
  4. A página indica onde este channel é usado para backup
