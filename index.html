Playlists
🎧

Arrasta ou clica para colar um link de playlist

open.spotify.com/playlist/…
Como usar
  1. No Spotify: ··· → Partilhar → Copiar link da música/playlist
  2. Cola aqui ou arrasta — título, artista e capa preenchidos automaticamente
  3. Para GIF: no Discord, botão direito no GIF → Copiar endereço da imagem
  4. O link deve começar com media.discordapp.net

Spotify API

Credenciais para auto-fetch de músicas e playlists.

Como obter
  1. Vai a developer.spotify.com/dashboard
  2. Cria ou abre a app existente
  3. Copia o Client ID e Client Secret
  4. Para usar conta com Premium: usa as credenciais dessa conta

Channels

Channels Discord para backup. Adiciona, edita e remove livremente.

Como obter Channel ID
  1. Discord: Configurações → Avançado → Activa Modo de Programador
  2. Botão direito no channel → Copiar ID do Channel
  3. Label = nome interno (ex: music, projects)
